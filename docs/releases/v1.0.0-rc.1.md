# ktui v1.0.0-rc.1

`v1.0.0-rc.1` is the release candidate for ktui's first stable major version. It is intended to validate the v1 CLI, configuration, export, installation, and update contracts before the final release.

## Highlights

- Cobra-based command structure with Bash, Zsh, Fish, and PowerShell completion.
- Multiple Komari connection profiles, including runtime switching and profile rename support.
- Sheet and line views, node details, history and ping charts, search, sorting, filtering, alerts, mouse support, and runtime settings.
- JSON, CSV, and Markdown exports plus checksum-verified self-update support.
- Detailed help compatibility for existing `ktui help <topic>` workflows.
- CI gates for tests, vet, race detection, GoReleaser configuration, and all six release build targets.

## Compatibility

No configuration migration is required from v0.5.0. Existing profiles and settings remain supported. Windows self-update still downloads the replacement executable and requires manual replacement because a running `.exe` cannot replace itself.

Report any crash, configuration loss, installer failure, incorrect export, or CLI incompatibility before `v1.0.0` is promoted.
